## Purpose of This Guide

This guide provides Hiring Managers, Supervisors, and Department Leaders with a clear step-by-step process for submitting and managing employee changes in BambooHR. These changes ensure proper updates to payroll, benefits, IT systems, building access, job titles, reporting structure, and compliance documentation.

## Employee Changes Requiring Manager Action

Managers should submit an employee change request for changes such as:

- Position or title change
- Pay rate or salary adjustment
- Bonus to be paid
- Promotion or demotion
- Department change
- Reporting manager change
- Employment status change such as full-time or part-time
- Work schedule or shift change
- Location change such as onsite, remote, or hybrid
- Transfer between divisions such as AAP to API or API to AAP
- Offboarding or termination process

## Initiate the Employee Change in BambooHR

To submit an employee change request:

1. Log in to BambooHR.
2. Go to the employee's profile.
3. Click **Request a Change** at the top of the page.
4. Select the appropriate change type.
5. Complete all required fields.
6. Add comments or attached documents when needed, such as a job description or justification.

**Common BambooHR Request Types:**

- **Job Information -** Position/title changes, department changes, reporting manager changes, work schedule changes, location changes, transfers between AAP and API, and demotions involving job changes.
- **Compensation -** Pay rate or salary adjustments, including pay changes related to a demotion.
- **Bonus to be Paid -- Payroll Notification -** One-time bonus payments
- **Promotion -** Job/title change and pay change together
- **Employment Status -** Full-time or part-time status changes and terminations

**Note:** Address changes should be submitted directly by the employee in BambooHR and are not part of the manager employee change process.

:::callout warning "Important"
Fields marked with an asterisk are required. Incomplete requests may delay HR, payroll, IT, or facilities updates.
:::

## Enter the Required Change Details

Include all details that apply to the requested change:

- Effective date of the change
- Reason for the change
- New position title, if applicable
- Updated pay rate or salary, if applicable
- New department or reporting manager, if applicable
- New schedule, shift, or work location, if applicable
- Change in full-time or part-time status, or scheduled hours, if applicable
- Transfer between AAP and API, if applicable
- Any related access, equipment or workspace needs

**Note:** If any acces, equipment, or workspace needs to be changed, the manager must update the Position Request

## Change-Specific Guidance

### Pay Rate or Salary Change

For compensation-related changes:

- Select Compensation for pay-only changes
- Select Promotion if the employee is receiving both a title change and a related pay change as part of a promotion
- Enter the effective date and updated pay information
- Include justification for the change in BambooHR
- HR reviews the request for compliance and payroll processing

:::callout warning "Important"
Submit payroll-related changes before payroll cutoff.
:::

### Position or Title Change

When an employee's position or title changes:

- Update the official job title
- Provide an updated job description, if applicable
- Confirm whether the change affects the org chart placement, reporting structure or system access
- Update the Position Request if access, equipment or workspace needs change

HR updates the organizational chart and coordinates related system changes as needed.

### Division, Department or Manager Change

When an employee moves to a new division, department, or reporting manager:

- Select the new division, if applicable
- Select the new department, if applicable
- Assign the new reporting manager, if applicable
- Document any related workflow or access changes
- Update the Position Request if access, equipment or workspace is changing

HR updates evaluation routing, workflows, org chart placement, and other related system settings as needed.

### For transfers between AAP and API:

Transfers between divisions require additional coordination because they may affect payroll, systems, email, permissions, branding details, and workspace setup.

- HR approval is required
- IT updates email structure, network access and permissions
- Marketing updates the employee's email signature template
- Facilities adjusts workspace and badge access, if applicable

### Schedule, Shift, or Location Change

When an employee's schedule, shift, or work location changes:

- Update the new schedule, shift, or work location
- Document any changes to hours for payroll accuracy
- Note whether the employee is working onsite, remote, or hybrid, if applicable
- Update the Position Request if access, equipment, or workspace needs change

If the location changes:

- HR updates records as needed
- IT updates login, access, or location-based settings, if applicable
- Facilities updates workspace or building access, if applicable

### Employment Status Change

When an employee changes from full-time to part-time or part-time to full-time:

- Update the employment status in the request
- Confirm any changes to scheduled hours
- Promptly communicate the change to HR, since changes in full-time or part-time status may affect legally mandated benefits and other benefit eligibility
- HR reviews the change and coordinates any related benefits or payroll updates

## Manager Responsibilities for Documentation

Managers are responsible for making sure the request is complete, timely, and clearly documented.

Manager responsibilities include:

- Submitting the change request before the effective date
- Attaching updated job descriptions when applicable
- Documenting the rationale clearly in BambooHR comments
- Communicating changes to affected team members

## HR Responsibilities After Manager Submission

After the manager submits the request, HR is responsible for processing and routing the change.

HR responsibilities include:

- Verifying the accuracy of the request
- Confirming alignment with policies and pay bands
- Applying payroll changes
- Notifying IT, Facilities, Marketing, and Finance when needed
- Updating the employee file and organizational chart

## IT and Facilities Workflow Triggered by Changes

Some requests trigger operational follow-up after HR review.

- Email or location changes: IT creates updated credentials
- Department or manager changes: submit an access update request for shared drive permissions while shared drives remain in use
- Location changes: Facilities updates workspace or desk assignment and coordinates badge access when applicable
- New equipment needs: IT tickets are generated automatically

:::callout info
Following the SharePoint and Intune rollout, drive access updates will move from shared-drive requests to SharePoint-based access control.
:::

## Employee Notification

Once the change is approved and processed:

- HR sends the official change confirmation to the manager who requested the change
- The manager discusses expectations and new responsibilities with the employee
- The employee receives the updated job description when applicable
- The employee should acknowledge schedule or pay changes when applicable

## Termination or Offboarding (Brief Overview)

If the employee change involves separation:

- Manager notifies HR immediately
- Submit the Offboarding Request in BambooHR
- HR processes final pay, benefits, and exit steps
- Manager collects company property such as keys, badges, and equipment and returns items to Facilities or IT
- IT disables accounts and removes system and access privileges

For more information, see the Manager Offboarding Process Guide.

## Timeline Expectations

- Submit change requests at least 5 business days before the effective date
- Payroll-related changes must be submitted before payroll cutoff
- Department or location moves usually require 5 to 10 days for IT and Facilities preparation
