## Recruiting Workflow in BambooHR (HRA-02)

Your workflow is **triggered by supervisor status updates** in BambooHR. Monitor candidate status changes daily.

**Step-by-step:**

1. **"Schedule Interview"** status → call the applicant to coordinate date/time
2. Send confirmation email: search applicant (upper-right search bar) → open screening page → Emails → correct template → send (include date, time, location)
3. **"Candidate Selected For Hire"** status → contact candidate with a verbal offer
4. Accepted verbal offer → open profile → status drop-down → **Create Offer Letter** → correct template → complete all details → send for signature
5. Offer letter signed → initiate background and drug screening immediately (HRA-03)
6. Complete **reference checks** (RM-01) — see below
7. Screening results received + clear → schedule start date (**Tuesdays preferred**)
8. Send New Hire Packet via BambooHR (correct template + state tax forms as needed)
9. Confirm all pre-hire screenings and reference check docs stored in BambooHR

:::callout warning
Escalate immediately if: candidate declines after verbal acceptance, offer terms need exception, background/drug screen returns concern, reference indicates risk, or candidate requests an accommodation.
:::

## Background & Drug Screening (HRA-03)

**Triggered by:** Signed offer letter — initiate immediately. Never confirm a start date until results are received.

**Universal rules:**

- Drug tests are **free**
- Must be completed within **3 business days** of offer letter signature
- Applicant must bring a **photo ID**
- Update BambooHR status to **"Pending Drug Screening Results"** immediately

**Memphis:** Order → Order w/ Invitation → Standard → A La Carte → **(AF) LabCorp – 10 Panel** → Next → Background Check Authorization template → Send Invitation. Applicant receives email to authorize background check and access testing locations.

**Scottsboro:** Send APIS Drug Screening Instructions template from BambooHR. Testing: **The Family Life Center**, 211 S. Market Street, Scottsboro, AL 35768 (Mon–Fri 8am–5pm, closed 12–1pm for lunch). In Employvio: Order → Order → Standard → Next → **Standard Screening – 1 County** → Submit.

**Results:** Memphis → Employvio PDF | Scottsboro → Phoenix LIS (Proton Pass / HR Manager login)

:::callout warning
Escalate immediately if: cannot test within 3 days, site inaccessible, results delayed, potentially positive/failed, identity mismatch, or any irregularity.
:::

## Reference Checks (RM-01)

**Minimum:** **3 successful contacts** — at least 1 employer reference + at least 1 personal reference.
(Applicants typically provide ~6 references; you are selecting 3 to contact.)

**Have ready:** Your full name, callback number, and the position applied for.

**Employer core questions:** Dates of employment | Attendance and punctuality | Work quality and reliability | Teamwork and communication | Eligible for rehire? (if no longer employed there)

**Warehouse add-ons (select 1–3):** Productivity/accuracy | Safety mindset | SOP compliance | Shift reliability | Equipment experience

**Administrative add-ons (select 1–3):** Attention to detail | Handling confidential info | Communication skills | Managing competing priorities | Professionalism

**Personal reference core questions:** How long known and how? | Reliability and follow-through | Handling stress/feedback | Communication and professionalism | Any job-related concerns?

:::callout warning
Never ask or document: Medical/disability, age, religion, family status, pregnancy, national origin, or any protected characteristic.
:::

**Documentation:** Reference name + phone | Date contacted | Your full name | Concise job-related summary

**Filing:** Save as `YYYY-DD-MM - Reference Check - FirstName LastName` → upload to BambooHR > Resumes, Applications and Pre-Hire Records → mark task complete in BambooHR

**Escalate if:** Cannot reach 3 references | Serious concerns (safety, theft, harassment, dishonesty, policy violations) | Candidate ineligible for rehire | Info conflicts with application
