## Purpose of This Guide

This guide outlines the steps managers must follow when offboarding an employee to ensure timely system access removal, proper notifications, and compliance with AAP/API policies.

## Quick Reference: Offboarding Request Process Checklist

Use the Offboarding Request Process Checklist as a one-page quick reference alongside this guide. It serves as a visual reminder of the Immediate and Scheduled Offboarding paths in BambooHR and the workflow triggerd after submission.

**Checklist highlights:**

- Log into BambooHR, search the employee, open their profile, and go to the **Offboarding Request** tab.
- For **Immediate Offboarding,** select **"Yes, disable access immediately"** and choose the required tasks.
- For **Scheduled Offboarding,** enter the required dates and select the required tasks.
- Submitting the request creates an **IT Service Request Ticket.** Immediate Offboarding also sends an email notification to IT and other applicable system owners.

## 1. Access the Offboarding Request (BambooHR)

Log in to BambooHR, search for the employee, open the employee's profile and select the **Offboarding Request** tab.

## 2. HR Approval Required Before Termination

Before initiating any termination (voluntary or involuntary), the manager must notify and involve HR in advance. For involuntary terminations, HR must be involved before the termination occurs.

This helps ensure:

- Compliance with federal and state employment laws
- Appropriate review of documentation, performance history, and prior corrective action
- Confirmation that the decision is consistent, job-related, and legally defensible

Managers must provide HR with supporting documentation, as applicable, including:

- Coaching, counseling, and/or corrective action documentation
- Attendance records
- Performance documentation, such as examples, metrics, recent issues, or patterns of concern
- Relevant emails or notes previously shared or discussed

:::callout info
HR may already have some or all of this documentation on file. However, managers are responsible for ensuring HR has the most current and complete information before a termination is finalized, including the specific documentation supporting the decision. This documentation may be needed for unemployment claims, legal inquiries or internal review.
:::


## 3. Immediate Offboarding (Same-Day / Immediate Access Removal)

Use this option when the employee is separating immediately and access must be disabled right away. Immediate Offboarding must be used for involuntary terminations.

**Steps:**

1. Select **Immediate Offboarding**.
2. In the dropdown, indicate whether access must be disabled immediately.
3. If **Yes**, IT will be notified to disable access right away.
4. Select the required **Offboarding IT Tasks.**
5. Add any notes needed for IT, such as email forwarding, shared mailbox details, equipment instructions, or other special access needs.

**What happens next (automatically):**

- An **IT Service Request Ticket** is created.
- An email notification is sent to IT and other applicable system owners based on division and location.

:::callout warning
If an after-hours or safety/security situation requires immediate access removal, submit Immediate Offboarding and contact HR as soon as possible.
:::

## 4. Scheduled Offboarding (Planned Last Day)

Use this option when the employee's last day is known in advance (resignation, retirement, planned transition, etc.)

**Steps:**

1. Enter the **Scheduled Last Day of Employment**.
2. Enter the **date and time access should be disabled,** typically the end of the employee's last working day unless HR or IT directs otherwise.
3. Select all required **Offboarding IT Tasks**.
4. If email must be forwarded, enter the forwarding address.
5. Add any additional notes in the **Offboarding Request Notes** field, such as equipment return details, special access needs, or shared mailbox instructions.

**Available Offboarding IT Tasks may include:**

- Disable AD Account
- Forward Email
- Remove Adobe License
- Remove Dynamics License
- Remove Fax Number
- Remove MapiLab License
- Remove Power BI License
- Reset Email Password

If email must be forwarded, enter the forwarding address.

Add any additional notes in the **Offboarding Request Notes** field (e.g., equipment return, special access, shared mailbox needs).

**What happens next (automatically):**

- An IT Service Request Ticket is created.
- Offboarding tasks are assigned in BambooHR by division and location.

## 5. Manager Responsibilities

Managers are responsible for making sure offboarding details are accurate, complete, and timely. 

**Manager responsibilities include:**

- Ensuring all offboarding selections are accurate, since the details entered determine IT ticket routing.
- Providing clear notes for IT regarding special instructions, such as shared mailboxes, forwarding needs, critical access removal.
- Communicating with HR immediately if offboarding circumstances change, including changes to date, time, reason, or urgency.
- Collecting company property as required and coordinating return logistics.
- Confirming that termination and offboarding steps align with HR guidance.

## 6. Important Reminders

- HR approval should be obtained before any termination whenever possible.
- Immediate offboarding must be used for involuntary terminations.
- Access removal is time-sensitive for security and compliance.
- Scheduled access disable time should generally be the end of the employee's last working day.
- Clean documentation matters after the employee leaves as well; termination support records may later be needed for unemployment claims, legal questions or internal review.
